## 关于

这个项目是 [bazel remote execution](https://github.com/bazelbuild/remote-apis) 的最小实现，可以用来学习了解 bazel 的远程执行时如何工作的。我也期望这个项目能够逐步完善，并且可以在生产环境中被使用

## 开发

修改 `cmd` 和 `pkg` 中的代码, 运行如下脚本

```bash
./deploy/docker-compose/up.sh
```
